// Принцип сортировки объявлений в списке
Каждое объявление имеет 5 доп. полей:
 - Кол-во дней размещения
 - Кол-во поднятий в поиске 
 - Премиум статус
 - Выделение

При покупке: 
 - кол-ва дней размещения в поле "finished" записывается дата окончания показа
 - кол-ва поднятий в поле "lifts_num" записывается кол-во
 - Премиум статус поле "premium" записывается дата окончания опции
 - Выделение поле "selected" записывается дата окончания опции

Механизм работы:
Крон будет запускаться например каждый (час???) и он будет проверять купленные 
опции для текущего объявления и на основе их данных обновлять поля для самого 
объявления (если это необходимо).

 - VIP статус не обязательно показывать в таблице объявления, т.к. он на выдачу 
не влияет, а показывает объявление в отдельном блоке.

 - Кол-во дней размещения обновляется один раз и до момента покупки новой опции,
т.к. поле "finished" фильтруется и оно уже не будет попадать в список. Удалять 
старые объявления через ??? дней.

 - Премиум статус обновляется один раз до момента следующей покупки опции.
Если объявление премиум оно исключается из обычного спсика и попадает в премиум список.

 - Выделение обновляется один раз до момента следующей покупки опции.
